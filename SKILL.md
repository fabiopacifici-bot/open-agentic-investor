# Open-Agentic-Investor

## Description
Open-Agentic-Investor is an OpenClaw skill designed to simplify investment portfolio management. It combines portfolio analysis, buy/sell recommendations, and recovery strategy suggestions. The skill integrates seamlessly with Trading 212 to optionally execute trades programmatically.

## Key Features
- **Portfolio Analysis**: Review current stock positions and evaluate buy/sell signals.
- **Dynamic Strategies**: Offers personalized risk-adjusted recommendations based on thresholds and user-defined configurations.
- **API Integration**: Automated API prompts and validations during setup.
- **Platform Integration**: Aligned with OpenClaw Hub standards for easy deployment.

## Installation
To install this skill via ClawHub, use the command:
```bash
clawhub install open-agentic-investor
```

## Usage Flow
1. **Setup**: Provide API credentials during installation.
2. **Portfolio Review**: Fetch and analyze stock data.
3. **Actionable Insights**: Receive recommendations for cost-averaging, diversification, or rebalancing.
4. **Automate (Optional)**: Integrate with Trading 212 for proactive order execution.
